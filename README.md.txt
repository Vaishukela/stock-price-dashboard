# Stock Price Dashboard (Streamlit)

Interactive finance dashboard to explore live stock prices, daily returns and trading volume.

## Features
- Ticker input (comma separated)
- Time period filter
- KPIs
- Price trend, daily returns and volume charts

## Tech Stack
Python, Pandas, Streamlit, Plotly, yfinance

## Data Source
Yahoo Finance (via yfinance)

## Run locally
python3 -m pip install -r requirements.txt
python3 -m streamlit run app.py
